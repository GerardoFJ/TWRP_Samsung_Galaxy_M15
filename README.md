# TWRP for Samsung Galaxy M15 5G (SM-M156B)

A custom build of TWRP (Team Win Recovery Project) and installation roadmap for the Samsung Galaxy M15 5G.

## Overview

This repository contains documentation, guides, and resources for installing custom GSI ROMs on the Samsung Galaxy M15 5G (SM-M156B) device. The primary focus is on building and implementing a functional TWRP boot menu.
